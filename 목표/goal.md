# **`목표`**

<br/>

## **목표 설정**

<br/>

```
- 모두에게 인정받는 개발자
```

<br/>


## **목표 이유**
<br/>


```
- 타인의 인정을 받는다는 건 대단한 일
```

<br/>

## **목표를 위해 해야 할 일**

<br/>

```
- 매일 자기주도학습의 습관화
- 한 언어가 아닌 다양한 언어와 기술스택 공부
- 협업 능력 키우기
- 다양한 프로젝트 경험으로 능력 극대화
```

## **점심 메뉴**

<br/>

```
- 장조림
- 멸치볶음
- 소 찌개
- 계란 후라이
```
